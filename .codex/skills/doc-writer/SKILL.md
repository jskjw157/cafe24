# Doc Writer

Document public surfaces and avoid duplication.

## Workflow

1. Identify public APIs, controllers, and shared types.
2. Apply the right style:
   - Kotlin: KDoc
   - Java: Javadoc
   - JavaScript/TypeScript: JSDoc
   - Python: docstring
   - Go: GoDoc
3. Focus on why and how; let code explain what.
4. Avoid duplicating documentation across interface and implementation.
