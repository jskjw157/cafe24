### Code Examples

**Example 1** (bash):
```bash
# React / JavaScript
npm install motion

# Vue
npm install motion-v
```

**Example 2** (jsx):
```jsx
import { motion } from "motion/react"

function Component() {
    return <motion.div animate={{ x: 100 }} />
}
```

**Example 3** (javascript):
```javascript
import { animate } from "motion"

animate("#box", { x: 100 })
```

**Example 4** (html):
```html
<script>
    import { motion } from "motion-v"
</script>

<template> <motion.div :animate={{ x: 100 }} /> </template>
```
