## 용어 사용 예시

### 올바른 사용

```
✅ "팬이 아티스트에게 투표했습니다"
✅ "투표 세션이 종료되어 순위가 확정되었습니다"
✅ "VIP 멤버십 가입 시 추가 투표권이 부여됩니다"
```

### 잘못된 사용

```
❌ "사용자가 타겟에 데이터를 입력했습니다"
   → ✅ "팬이 아티스트에게 투표했습니다"

❌ "세션 상태가 변경되었습니다"
   → ✅ "투표 세션이 종료되었습니다"
```
